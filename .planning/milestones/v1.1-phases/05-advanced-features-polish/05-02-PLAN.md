<task type="auto" tdd="true">
  <name>Task 1: Field selection utility and JSONL helpers with tests</name>
  <files>src/lib/fields.ts, tests/unit/fields.test.ts</files>
  <behavior>
    - pickFields({ id: 1, text: "hi", date: "2026-01-01" }, ["id", "text"]) returns { id: 1, text: "hi" }
    - pickFields({ id: 1, media: { filename: "a.jpg", fileSize: 100 } }, ["id", "media.filename"]) returns { id: 1, media: { filename: "a.jpg" } }
    - pickFields(obj, ["nonexistent"]) returns {} (silently omits invalid fields)
    - pickFields(obj, ["nested.deep.field"]) with missing intermediate returns {} (no crash)
    - applyFieldSelection({ messages: [{ id: 1, text: "hi" }], total: 5 }, ["id"]) returns { messages: [{ id: 1 }], total: 5 } (metadata preserved)
    - applyFieldSelection({ status: "ok" }, ["status"]) returns { status: "ok" } (non-array data passes through with field filter on scalar entries)
    - extractListItems({ messages: [msg1, msg2] }) returns [msg1, msg2]
    - extractListItems({ chats: [c1] }) returns [c1]
    - extractListItems({ members: [m1] }) returns [m1]
    - extractListItems({ topics: [t1] }) returns [t1]
    - extractListItems({ files: [f1] }) returns [f1]
    - extractListItems({ status: "ok" }) returns null (not a list command)
    - extractListItems(null) returns null
  </behavior>
  <action>
    Create src/lib/fields.ts with three exported functions:

    1. `pickFields<T>(obj: T, fields: string[]): Partial<T>`:
       - For each field in fields, split by '.' to get path parts
       - Walk the object along the path; if any intermediate is null/undefined/non-object, skip this field (silently omit)
       - If value found, reconstruct the nested path in the result object
       - Return result cast as Partial<T>

    2. `applyFieldSelection(data: unknown, fields: string[]): unknown`:
       - If data is null or not an object, return data unchanged
       - Iterate entries of data. For each entry:
         - If value is an array with objects (length > 0, typeof [0] === 'object'), apply pickFields to each item (this is the items array)
         - Otherwise, preserve the value as-is (metadata like total, count)
       - Return the reconstructed object
       - This preserves the data shape: { messages: [{filtered}], total: N }

    3. `extractListItems(data: unknown): unknown[] | null`:
       - If data is null or not an object, return null
       - Check known array properties in order: messages, chats, members, topics, files
       - If found and is an Array, return it
       - Otherwise return null

    Create tests/unit/fields.test.ts with comprehensive tests per behavior list above.
  </action>
  <verify>
    <automated>npx vitest run tests/unit/fields.test.ts -x</automated>
  </verify>
  <done>pickFields handles flat and dot-notation paths, silently omits missing fields. applyFieldSelection preserves metadata alongside filtered array items. extractListItems detects all known list shapes. All tests pass.</done>
</task>
