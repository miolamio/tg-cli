    New module with:

    - `FILTER_MAP`: Record mapping 8 filter names to factory functions returning `Api.InputMessagesFilter*` instances (photos, videos, documents, urls, voice, music, gifs, round). Use factory functions `() => new Api.InputMessagesFilterPhotos()` etc. to create fresh instances per call.
    - `VALID_FILTERS`: string array of valid filter names (`Object.keys(FILTER_MAP)`).

    - `detectFileType(ext: string): 'photo' | 'video' | 'voice' | 'document'` — classify by extension for uploads. Map: .jpg/.jpeg/.png/.gif/.bmp/.webp -> photo, .mp4/.mov/.avi/.mkv/.webm -> video, .ogg/.opus -> voice, everything else -> document.
    - `formatBytes(bytes: number): string` — human-readable file size. <1024 -> NB, <1MB -> NKB (rounded), <1GB -> N.NMB, else N.NGB.
