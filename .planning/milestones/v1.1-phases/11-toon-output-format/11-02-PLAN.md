  <action>
    Create `tests/unit/toon-benchmark.test.ts`:

    ```typescript
    import { describe, it, expect } from 'vitest';
    import { countTokens } from 'gpt-tokenizer';
    import { encodeToon } from '../../src/lib/toon.js';
    ```

    For each fixture file:
    1. Import the fixture JSON (use `import ... from '../fixtures/toon-benchmark/X.json' with { type: 'json' }` or `JSON.parse(readFileSync(...))` -- use readFileSync since JSON imports may need special tsconfig config)
    2. Wrap in envelope: `{ ok: true, data: fixture }`
    3. Encode as JSON: `JSON.stringify(envelope)`
    4. Encode as TOON: `encodeToon(envelope)`
    5. Count tokens for both: `countTokens(jsonStr)`, `countTokens(toonStr)`
    6. Calculate savings: `1 - toonTokens / jsonTokens`
    7. Log savings for visibility: `console.log(\`${name}: JSON=${jsonTokens} TOON=${toonTokens} savings=${(savings*100).toFixed(1)}%\`)`
    8. Assert: `expect(savings).toBeGreaterThanOrEqual(threshold)`

    Test structure:
    - `describe('TOON benchmark gate')` with individual tests per fixture
    - Threshold: 0.20 (20%) for uniform array fixtures (messages, chats, profiles, search results)
    - Threshold: 0.15 (15%) for mixed-shapes fixture (non-uniform data, per Pitfall 5 in RESEARCH.md)
    - Each test is descriptive: `'achieves >= 20% token savings on 100+ messages'`
