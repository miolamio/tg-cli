## Tech Debt Summary

### Phase 2: Chat Discovery & Message Reading (6 items)
1. **senderName always null** — `serializeMessage(msg)` called without `senderEntity` in history and search
2. **Filter-pagination ordering** — `--type` filter applies after pagination slice in chat list
3. **Date-bounded pagination** — `--since` post-filters after gramjs pagination; `total` is unfiltered
4. **No date input validation** — invalid `--since`/`--until` silently produce NaN
5. **Join output missing chat.type** — returns `{id, title}` without `type` field
6. **MISSING_QUERY unreachable** — Commander `requiredOption` fires before handler guard

### Phase 1: Foundation & Auth (3 items)

2. **getOutputMode dead export** — exported but never imported by any production code
3. **12 unused type interfaces** — option interfaces defined in types.ts but commands use inline types
